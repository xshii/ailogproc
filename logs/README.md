# 日志文件目录

此目录用于存放您的日志文件。

## 日志文件命名

推荐使用时间戳命名：`logs_YYYYMMDD_HHMMSS.txt`

示例：
- `logs_20260207_153045.txt`
- `logs_20260208_091230.txt`

## 使用方法

1. 将日志文件放到此目录（按推荐格式命名）
2. 运行：`python main.py <Excel模板>`
3. 程序自动使用最新的日志文件

## 自动清理

程序运行时会自动清理旧日志，**只保留最新的5个日志文件**。

## 注意

- 此目录已在 `.gitignore` 中，日志文件不会被提交到 Git
- 如需示例文件，请查看 `examples/logs/` 目录
